spring源码阅读环境搭建

Posted on 2019-12-24

源码下载

spring的源码有两种下载途径：

  1. git clone
  2. release .zip格式压缩包下载
  • git clone：
    • 地址：https://github.com/spring-projects/spring-framework/tree/5.1.x
    • 选择最新的分支，clone到本地即可使用
  • release .zip格式压缩包下载：
    • 地址：https://github.com/spring-projects/spring-framework/releases
    • 解压下载好的zip文件并将获得的文件夹挪到适合的目录下即可

打包操作

  • 打开解压后获取到的文件目录，根目录下有import-into-idea.md及import-into-eclips.md两种导入源码阅读的指导文档

  • 这里由于我用的ide是idea，步骤参考idea指导md

    1. ./gradlew :spring-oxm:compileTestJava

    2. ./gradlew

    3. 我这里执行打包命令报错：

    4. 这个问题的原因是我使用的是下载的zip压缩包，包内无git仓库（.git文件），所以打包时无法找到对应的git命令，我们只需要在目录下执行：git init，git add *，git commit -m””即可

    5. 打包完成，继续根据md文档，导入到idea，这个比较久，我这里没出问题，在等待较长时间后自动完成了
